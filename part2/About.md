# 🧠 Hack Assembler (Nand2Tetris Project - Part 2)

## 📌 Overview

This project implements a **general-purpose assembler** for the Hack computer platform (Nand2Tetris).
The assembler converts Hack assembly language (`.asm`) into 16-bit binary machine code (`.hack`).

---

## 🎯 Objective

To design and implement a **two-pass assembler** that:

* Reads a `.asm` file
* Handles symbols (labels & variables)
* Converts instructions into 16-bit binary
* Matches the output of the official Nand2Tetris assembler

---

## ⚙️ Features

✔️ A-instruction support (`@value`, `@symbol`)
✔️ C-instruction support (`dest=comp;jump`)
✔️ Label handling `(LOOP)`
✔️ Variable handling (starting at RAM address 16)
✔️ Symbol table implementation
✔️ Two-pass assembly process

---

## 🧠 How It Works

### 🔹 Pass 1 (Label Handling)

* Scans the file
* Stores labels with their instruction addresses

Example:

```
(LOOP)
```

Stored as:

```
LOOP → address
```

---

### 🔹 Pass 2 (Translation)

* Converts instructions into binary
* Assigns memory to variables
* Generates final machine code

---

## 🔢 Instruction Types

### 1. A-instruction

```
@value
```

➡️ Converts to:

```
0 + 15-bit binary value
```

---

### 2. C-instruction

```
dest=comp;jump
```

➡️ Converts to:

```
111 + comp + dest + jump
```

---

### 3. Labels

```
(LOOP)
```

➡️ Used for jumps (no binary generated)
