## ▶️ How to Compile and Run

### 🔹 Compile (Windows)

1. Open Command Prompt
2. Navigate to your project folder:

```
cd C:\Users\chari\OneDrive\Desktop\nand2tetris\nand2tetris\projects\JACK-STRING-IMPLEMENTATION\Part1
```

3. Run the Jack Compiler:

```
C:\Users\chari\OneDrive\Desktop\nand2tetris\nand2tetris\tools\JackCompiler.bat .
```

✔️ This will compile all `.jack` files in the folder and generate `.vm` files.

---

